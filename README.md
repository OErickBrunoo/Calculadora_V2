# aula-de-segunda-calculadora
Repositório criado para a entrega do trabalho do curso de Algoritmos e Pensamento Computacional, primeiro semestre de Análise e Desenvolvimento de Sistemas na UDF, ministrado pela professora Karla Roberto Sartin.
Trata-se de uma implementação de refatoração e modularização à uma calculadora científica previamente desenvolvida em atividade, agora sendo necessário adicionar um histórico a esta versão anterior que pedia um código mais direto. Além disto, devemos utilizar arrays e structs para exercitar/exemplificar o conteúdo lessionado.
Determinadas implementações se apresentaram como desafios e, portanto, utilizamos da ajuda de agentes de Inteligência Artificial para chegar a algumas conclusões, tudo está bem explicado nos commits do arquivo Calculadora.c, esclarecendo justamente a forma como utilizamos este tipo de reforço. Os links para acesso aos prompts estarão anexados ao final deste README logo antes dos nomes dos participantes do grupo.
O "esqueleto" do código no arquivo Calculadora.C foi criado pelo aluno Erick Bruno, dispondo a maior parte das funções de cálculos e a estrutura principal do código que posteriormente foi modificado pelos alunos Ector Falcão (este que vos redige), Leonardo Teles e Talisom, onde o primeiro dirigiu os outros dois para realizar as edições e garantir a compreensão mútua. Nos primeiros commits é possível observar que a forma como a refatoração e a modularização do código se apresentava foi alterada, isto ocorreu devido divergência na visão do Ector e do Erick sobre como separar as responsabilidades das funções presentes no código, sendo que Ector é apegado (demais) a expressão "responsabilidade única". Para todos os efeitos, a excelência do segundo citado é tamanha e acabou enviando um código quase completo da calculadora que está evidenciado no arquivo CalculadoraExtra.c e, assim sendo, garantimos que ambas as visões sobre modularização/refatoração podem se apresentar.

Os prompts estão bagunçados devido um fluxo de trabalho desorganizado e motivado pela pressa.
